Sobre o sistema

O Amigo Fiel é desenvolvido como uma aplicação web em camadas, pensada para ser simples de instalar e estender. Abaixo um resumo técnico rápido:

  • Backend: Django 5.2.x (Python 3.13)
  • Banco de dados: PostgreSQL
  • Frontend: Django Templates, CSS responsivo (padrões simples, sem framework pesado)
  • Uploads e mídia: armazenados em MEDIA_ROOT, servidos em desenvolvimento via configuração de static/media
  • Autenticação e perfis: três perfis: Usuário Comum, ONG e Empresa

Quer contribuir ou rodar localmente? Consulte o repositório GitHub e o README.md para instruções.
